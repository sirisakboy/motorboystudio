# Component Specifications

This folder contains the **Functional Contracts** for all universal components. Each component must define:

1. **Props/Inputs:** Data needed to render/configure the component.
2. **Events/Outputs:** Actions/callbacks emitted by the component.
3. **Style Tokens:** Mapping of design system variables (e.g., --accent-racing-red) to component properties.
